# Quine とは？

- ソースコード自身を出力するようなプログラム<br/>
  (下のコードがまさに Quine)

```javascript
eval((s = "console.log('eval(s='+JSON.stringify(s)+')')"));
```

```console
$ node quine.js | node - | node -
eval(s="console.log('eval(s='+JSON.stringify(s)+')')")
```

- **特に世の中の役には立たない**。
- 意外と書けそうで書けないところが面白い。
